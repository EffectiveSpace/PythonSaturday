Создать программу с графическим интерфейсом "Конвертор валют" используя библиотеку 'easygui':
(можно сделать что-то своё, например приложение для скачивания "ИГР","фильмов" и т.д...)
Код программы должен делать следующее:
1.При запуске программы должно выходить окно приветствия (например:Вас приветствует конвертор валют!)
2.После пользователю предлагается выбрать нужное действие (например: выбрать одну из валют (DOLL, EUR))
3.Когда пользователь выберет нужную валюту, должно открыться окно для ввода количества рублей.
4.После нажатия кнопки "Конвертировать" должно выйти окно с расчётом в виде: 10000р = 132$
5.Когда пользователь нажмет на кнопку "ОК". Ему должно выйти окно сообщения "хочет ли он продолжить?"
6.Если хочет, то цикл запускается снова, если нет - просто выйти из программы.
